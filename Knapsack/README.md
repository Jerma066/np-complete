# Knapsack

`BruteForce` and `BranchAndBound` implemetation of knapsack problem.

## Build

mkdir build && cd build  
cmake .. && make -jnproc  

## Fuctional testing

cd build/tests/  
./knapsack_test  

## Perfomace testing

cd build/knapsack/  
./bench
